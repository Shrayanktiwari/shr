# JP Infra Developers - Real Estate Website

## Overview
A professional, fully responsive real estate website for JP Infra Developers. Built with pure HTML5, CSS3, and vanilla JavaScript without any frameworks or libraries (except Google Fonts for typography).

## Purpose
Showcase properties for sale and rent, highlight company strengths, and provide a contact channel for potential clients. The website features a classy monochromatic design (black, white, grey) with elegant orange (#FF8C00) highlights for a premium, modern look suitable for a high-end real estate builder company.

## Project Structure
```
/
├── index.html          # Main HTML file with all sections
├── style.css           # Complete CSS styling with responsive design
├── script.js           # JavaScript for animations and interactivity
├── images/             # Directory for local images (currently using Unsplash URLs)
├── .gitignore          # Git ignore file
└── replit.md           # This documentation file
```

## Features Implemented
- **Sticky Navigation**: Fixed header with smooth scroll to sections
- **Hero Section**: Full-width background with headline and CTA button
- **Buy Property Section**: Grid layout with 8 property cards
- **Rent Property Section**: Grid layout with 8 rental property cards
- **Why Choose Us**: 4 feature cards with icons and hover animations
- **About Section**: 2-column layout with company description and image
- **Contact Section**: Form with name, email, message fields and contact information
- **Footer**: Dark background with copyright and social media icons
- **Responsive Design**: Mobile-first approach with breakpoints for tablet and desktop
- **Smooth Animations**: Fade-in effects on scroll, hover animations on cards and buttons

## Design System
- **Colors**: 
  - Primary Orange: #FF8C00
  - Black: #0a0a0a
  - Dark Grey: #1a1a1a
  - Medium Grey: #333333
  - Light Grey: #666666
  - White: #ffffff
  - Background Light: #f5f5f5
- **Typography**: Poppins font family from Google Fonts
- **Spacing**: Consistent padding and margins using CSS custom properties
- **Shadows**: Soft shadows for depth (sm, md, lg variants)

## Technology Stack
- HTML5 (semantic markup)
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (DOM manipulation, event handling)
- Google Fonts (Poppins)
- Unsplash (placeholder images)
